# Research Essays

Non-project work — essays, explorations, and standalone pieces.

## Organization

```
research-essays/
├── essay-topic-1/
│   ├── README.md    # Goals and status
│   ├── draft.md     # Current draft
│   └── notes.md     # Research notes
├── essay-topic-2/
└── ideas/           # Early-stage ideas
```

## Usage

- For writing that isn't tied to a specific collaborative project
- Each essay can have its own folder with README/notes
- Move to `publications/` when complete

---

*For collaborative research, use `projects/`. For published work, use `publications/`.*
